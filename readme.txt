== Description ==

Monetize your WordPress website/blog with adcrun.ch shortlinks. This plugin converts your links to adcrun.ch links automatically.

You will able to:

*   Enable/Disable adcrun.ch feature
*   Convert either all links or only outgoing links to adcrun.ch
*   Choose ad type: Interstitial or Banner ad

    You will need an adcrun.ch id. To get it, login to adcrun.ch go to http://adcrun.ch/referrals.php and find ?r=xxxx. xxxx is your adcrun.ch ID.

== Installation ==

1. Go to Plugins-> Add New Plugin -> Upload
2. Upload `adcrun.ch.zip`
3. Activate the plugin.
4. Navigate to Settings-> adcrun.ch Settings, enable plugin and put your adcrun.ch id. To get it, login to adcrun.ch go to http://adcrun.ch/referrals.php and find ?r=xxxx. xxxx is your adcrun.ch ID.
